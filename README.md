# ☕ Estudos e Atividades em Java

Bem-vindo ao meu repositório de estudos! 🚀

Este espaço foi criado para documentar meu aprendizado, armazenar exercícios práticos e organizar as atividades da faculdade focadas em **Java**.

## 📚 Estrutura do Projeto

Os códigos estão organizados em pacotes (pastas) dentro de `src/`, separados pelos paradigmas de programação que estou estudando:

* **`poo/`**: Exercícios e projetos focados em **Programação Orientada a Objetos** (Classes, Herança, Polimorfismo, Encapsulamento, etc.).
* **`estruturada/`**: Atividades focadas em lógica de programação e **Programação Estruturada**.

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** Java
* **IDE:** IntelliJ IDEA (JetBrains)
* **Controle de Versão:** Git & GitHub
